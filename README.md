# vite-ember-ssr-server

Server-side rendering (SSR) node server library for Ember on Vite.

A replacement for [fastboot-app-server](https://www.npmjs.com/package/fastboot-app-server).

This is a prototype not yet published to npm, so just install via the latest hash
or fork it to help clean it up.

See [the example application](https://github.com/kmccullough/vite-ember-ssr-example) 
for further details.
